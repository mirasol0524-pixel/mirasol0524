# 學習的問題很古老｜HTML 簡報

依據 `source/source.docx` 製作的 10 頁靜態 HTML 簡報，主題是認知科學下的四種學習困難：動機、理解、記憶與遷移。

## 使用方式

直接以瀏覽器開啟 `index.html`。使用方向鍵、Page Up／Page Down、空白鍵或點擊畫面左右半部換頁；網址雜湊可直接定位頁次，例如 `index.html#4`。

## 專案結構

- `index.html`：簡報內容、樣式與互動，無外部相依套件。
- `source/source.docx`：使用者提供的原始附件副本。
- `tools/gh-2.96.0/bin/gh.exe`：專案內的 GitHub CLI 可攜版；未修改系統 PATH。
- `AGENTS.md`：後續維護規則。
- `TODO.md`：待辦事項。
- `PROGRESS.md`：進度與決策紀錄。

- 內容忠於來源，不引入未經查證的額外論點。
